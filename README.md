# 脊髄くも膜下麻酔 ブロック高シミュレータ｜離島麻酔ツールキット

比重・体位・投与量といった**制御可能因子**が、脊髄くも膜下麻酔（脊麻）のブロック高に「どの向きに・どの程度」効くかを、不確実性帯つきで示すオフライン対応の**教育用**ツール（PWA）。

- **公開URL**: https://nobumitsu-3141.github.io/spinal-level-sim/
- **構成**: 単一HTML（外部依存ゼロ）＋ Service Worker による完全オフライン
- **入力**: 比重（高/等/低）・体位・穿刺間隙・投与量・修飾因子（頭低位/妊娠/高齢/腹腔内圧↑）・**添加薬（髄注フェンタニル/モルヒネ）**
- **添加薬の効果**: 髄注オピオイドは**ブロック高をほぼ変えず**、効くのは**持続（特に術後鎮痛）**。フェンタニル＝術後鎮痛+約1–2h、モルヒネ＝術後鎮痛12–24h（遅発性呼吸抑制に警告）。根拠：Pöpping 2012(Pain)・Sultan 2016(Anesth Analg)・Choi 2000。
- **投与量**: 0.5%マーカイン想定で **mL と mg を併記**（0.5%=5 mg/mL）。スライダーは mL 基準。
- **用量プランナー（容量法）**: 手術部位の基本量（高比重）＋身長/体重/年齢補正で推奨開始量(mL/mg)を提案。添付文書の上限 4 mL(20 mg) で頭打ち。「投与量に反映」でスプレッド推定へ連携。
- **出力**: 推定ピーク感覚遮断レベル＋不確実性帯、ランドマーク到達、推定持続、デルマトーム図、臨床ノート
- **UX**: 教育プリセット（サドル/標準/高位/低比重）、リセット、設定＋結果のテキストコピー
- **対応**: iPhone / iPad / Mac（レスポンシブ・ホーム画面追加でオフライン起動）

## 用量プランナーについて（重要）
容量法のロジックは「麻酔科勤務医のお勉強日記」(2011) の実践的・経験的な個人の方法を元にしており、**形式的な引用文献はありません**。本アプリでは同記事の計算例3例（2.3 / 2.7 / 2.3 mL）と一致することを確認のうえ実装し、容量・上限は**マーカイン添付文書（0.5%高比重/等比重・アストラゼネカ：通常 2–4 mL/10–20 mg、上限 4 mL/20 mg）**で裏取りしています。あくまで開始量の目安であり、施設プロトコル・添付文書・最新ガイドラインでの確認が前提です。

## 使い方（オフライン）
1. iPhone / iPad / Mac で上記URLを一度開く
2. 「ホーム画面に追加」（iOS Safari）／インストール（Chrome）
3. 以後は電波がなくても起動・動作

## 更新方法
`index.html` 等を変更したら、`sw.js` の `CACHE` バージョン文字列（例 `spinal-level-v1.0.1`）と `index.html` のフッター表記を上げてからpush。古いキャッシュは自動削除され、次回起動で新版に更新される。

## エビデンス
- Carpenter RL, et al. *Anesthesiology* 1998;89:24-29（腰仙部CSF容量が最大の規定因子＝個別予測が困難な理由）
- Hocking G, Wildsmith JAW. *Br J Anaesth* 2004;93:568-578（制御可能因子は比重・体位・投与量。総投与量(mass)が効く）
- Greene NM. *Anesth Analg* 1985;64:715-730（比重と体位の相互作用）

---

> ⚠️ **注意**: 本ツールは**予測器ではなく教育用モデル**です。最終ブロック高の最大の規定因子（腰仙部CSF容量）は臨床的に測定不能で、同一手技でも数分節のばらつきが生じます。臨床判断・最新ガイドライン・施設プロトコル・添付文書での確認に代わるものではありません。
